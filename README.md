# imKontext

Backup operativo del proyecto `imKontext`.

## Qué es cada carpeta

- `imKontext/`: frontend actual que se publica como sitio estático.
- `backend/`: backend Node + Supabase y scripts de apoyo para generación de contenido.
- `EPUBS/`: material fuente local.

## Estado actual

- Producción actual: HTML/CSS/JS servidos estáticamente.
- API usada por el frontend: backend desplegado en Render.
- Base de datos: Supabase.

## Objetivo de este repo

Tener una copia de seguridad útil del frontend actual y del backend en evolución, sin mezclar archivos temporales, cachés locales o experimentos descartables.
